# UML Search MCP

An MCP server to search the `uml.edu` domain for up-to-date information. An important component of [unibot-v2](unibot-v2.umlcloudcomputing.org)

Written in Python. Uses [Docker](https://github.com/docker) and [MCP](https://modelcontextprotocol.io/docs/getting-started/intro). [MCPInspector](https://modelcontextprotocol.io/docs/tools/inspector) used for development.

Published container image available [here](https://github.com/UMLCloudComputing/uml-search-mcp/pkgs/container/uml-search-mcp)
Source code available on [GitHub](https://github.com/umlcloudcomputing/uml-search-mcp.git).
